ALGORITHM OVERVIEW
================================================================================

TRAIT SCORE CALCULATION:
- 20 questions × multiple trait responses
- Each response contributes to trait scores
- Normalized to 0-100 scale
- Generates RIASEC profile (6 trait scores)

CAREER MATCHING FORMULA:
Match Score = (TraitMatch × 0.5) + (AcademicFit × 0.3) + (SkillMatch × 0.2)

- Trait Match: Compare user's RIASEC profile with career requirements
- Academic Fit: User score vs. career prerequisites
- Skill Match: User's skills vs. career's required skills

Result: Top 5 career matches with 0-100% match scores

ROADMAP GENERATION:
- Identify skill gaps vs. required skills
- Distribute gap closure across 6 months
- Create monthly milestones
- Generate resource recommendations
- Track completion status

SKILL GAP ANALYSIS:
- Current skill level (0-100)
- Required skill level (typically 85)
- Gap size = Required - Current
- Timeline = Gap / learning velocity
- Curated resources for closure
