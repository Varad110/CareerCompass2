GETTING STARTED
================================================================================

QUICK START:

1. Install dependencies:
   $ pnpm install

2. Run development server:
   $ pnpm dev

3. Open browser:
   http://localhost:3000

4. Explore pages:
   - Landing page
   - Login/Signup
   - Dashboard
   - Quiz
   - Results
   - Profile
   - Resources

5. Read documentation:
   - START_HERE.md (navigation)
   - PROJECT_SUMMARY.md (overview)
   - README.md (full details)

FOR DATABASE:

1. Create database:
   $ mysql -u root -p < scripts/init-db.sql
   $ mysql -u root -p < scripts/seed-data.sql

2. Verify tables:
   $ mysql -u root -p career_compass
   > SHOW TABLES;

3. Check data:
   > SELECT * FROM career_profiles;
   > SELECT * FROM quiz_questions LIMIT 5;
